# TensorflowNow

演练Tensorflow项目。方便工程师了解和实现AI导向的产品，以便学术研究能够产品化。喜爱这个项目请捐助！！

## 安装系统依赖

> 使用pyenv进行Python版本管理
```
brew install pyenv poetry
pyenv install 3.10.13
pyenv global 3.10.13
pyenv local 3.10.13

poetry config virtualenvs.in-project true --local
```

## 安装项目依赖

> 使用poetry进行依赖管理

```
poetry instsall
```
